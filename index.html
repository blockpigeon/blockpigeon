BlockPigeon

Inter-blockchain event transfer oracle


How it works

  1. Emit an event from your contract.
    Blockpigeon will only deliver a validly emitted event from one contract on a supported chain to another contract on a supported chain.
  2. Next look up the contract address of the pigeon on your chain in the table below.
    Destination Chain ID Postage Contract
    Ethereum 1 ~ $25 0x785d28042abe9b7968966a61db44fb81d31ba85a
    BSC 56 ~ $2.5 0xaf50d7460B29aef8757ad9f08AA709269e4bf9c3
    xDai 100 ~ $2.5 0x0E1AEAfFE99Ecc2D177C59D1587a410709E760CF
    Fuse 122 ~ $2.5 0x0576B45F588424AA14F1B18A4D9fD3C260AD6A2a
    Polygon 137 ~ $2.5 0xCd875A2dcb17368D7233BD8159A0a7C68fBba621

  3. Using the pigeon on your (source) chain query the cost to deliver the event to the destination chain. This will return the cost in the smallest unit of the native currency on your source chain (e.g. wei).

    function pigeonCost(uint256 _dest_chain_id)
    returns (uint256 pigeon_call_cost)

  4. Send the required payment to the pigeon along with the event ID, destination chain ID, and destination contract ID and the source txn hash (or 0 if you are calling the pigeon from the transaction where the event is being generated).

    function pigeonSend (
        uint256 _source_txn_hash,
        uint256 _source_event_id,
        uint256 _dest_chain_id,
        uint256 _dest_contract_id
) returns (bool success)

  5. The blockpigeon oracle will deliver the specified event to the following function on the specified destination contract

    Pigeon Oracle (sender):

    0x94175c8880a131605a72e61a27ff1568876fabe2

    function pigeonArrive (
        uint256  _source_chain_id,
        uint256 _source_contract_id,
        uint256  _source_block_no,
        uint256  _source_confirmations,
        uint256 _source_txn_hash,
        uint256 _topic0,
        uint256 _topic1,
        uint256 _topic2,
        uint256 _topic3,
        uint256 _topic4,
        uint256 _topic5
) returns (bool success)

Over time additional chains will be added. Anyone may use the BlockPigeon provided they pay the carrier fee. The Pigeon Oracle is redundantly provided from several locations around the world.
